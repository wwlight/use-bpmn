# use-bpmn

## 说明

- 本项目是基于 [bpmn.js](https://github.com/bpmn-io) 和 [nuxt](https://nuxt.com/) 开发演示
- 涵盖编辑器、属性面板及汉化等
- 覆盖的场景：
  - [x] 读取属性面板数据
  - [x] 更新属性面板数据
  - [x] 属性面板添加自定义事件
  - [ ] 属性面板添加属性
  - [ ] 自定义属性面板

## 命令

```bash
pnpm install

pnpm dev

pnpm generate
```
