# use-bpmn

## 说明

- 本项目是 [bpmn.js](https://github.com/bpmn-io) 在 [nuxt](https://nuxt.com/) 中实践
- 涵盖编辑器、属性面板及汉化等

## 运行

```bash
pnpm install

pnpm dev

pnpm generate
```
